# soccermatics

A Python project for exploring and analyzing football (soccer) data using StatsBomb event data. This repository demonstrates various techniques for visualizing and analyzing soccer metrics using `mplsoccer`, `matplotlib`, and other analytical tools.

## Notebooks

- `plotting_shots.ipynb` - Example notebook demonstrating shot visualization and basic match analysis.

## Requirements

- Python 3.x
- `mplsoccer`
- `matplotlib`
- `numpy`
- `pandas`

## Usage

1. Install the required packages in your environment.
2. Explore the notebooks to learn different soccer analytics techniques.
3. Modify and extend the code to analyze your own match data.

## About

This repository is inspired by the book "Soccermatics" and serves as a platform for learning and experimenting with football data analysis, visualization, and metrics.
